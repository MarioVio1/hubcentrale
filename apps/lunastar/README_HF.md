# 🎮 GameHub - Piattaforma Giochi Multiplayer

Una piattaforma di giochi multiplayer online con supporto per TV + Smartphone come controller.

## Giochi disponibili

- 🔴 **Forza 4** - Connetti 4 in fila
- 🃏 **Briscola** - Carte trevisane
- 🎴 **UNO** - Il classico colorato
- 🪙 **Scopa** - Prendi tutte le carte
- 🔍 **Indovina Chi** - Indovina il personaggio
- 📝 **Nome Città** - Cose, Animali, Città
- ♛ **Dama** - Il classico da tavolo
- 🎪 **Mercante in Fiera** - Vinci il jackpot

## Come giocare

1. Scegli un gioco
2. Inserisci il tuo nome
3. Condividi il codice stanza con gli amici
4. Gioca insieme!

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript
