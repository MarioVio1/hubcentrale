
BESTI DI VENETIA - Starter Pack

Questo pacchetto contiene una struttura base per creare un gioco basato su Tuxemon.

Cartelle principali:
mods/venetia/maps -> mappe Tiled
mods/venetia/monsters -> creature JSON
mods/venetia/npc -> personaggi
mods/venetia/items -> oggetti
mods/venetia/events -> eventi storia

Apri le mappe con Tiled Map Editor.
Aggiungi sprite e asset grafici nelle cartelle appropriate.

Questo è solo uno starter kit per iniziare lo sviluppo.
